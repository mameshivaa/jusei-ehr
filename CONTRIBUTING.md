本プロダクトはソースコード非公開の無料配布版です。外部からのプルリクエストやフォークによる開発参加は受け付けていません。

## バグ報告・機能要望

- GitHub Issues は公開していません。配布時に案内されるサポート窓口（メールまたは連絡フォーム）からご連絡ください。
- 報告時は「発生事象」「再現手順」「利用環境（OS/ブラウザ）」「スクリーンショット（可能なら）」を添えてください。

## コード変更

- 開発は専任チームが内部リポジトリで実施します。外部からのコード提供・パッチは受領していません。

## 行動規範

- 本プロダクトの利用・問い合わせに際しては、常識的な礼節と法令遵守をお願いします。コミュニティガイドラインやOSS向けのコントリビューション規約は適用されません。

## 開発環境メモ（社内向け）
