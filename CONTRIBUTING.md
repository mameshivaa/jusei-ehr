# コントリビューションポリシー（Source-Available版）

本プロジェクトは source-available で公開しています。Issue / Pull Request による改善提案を歓迎します。

## 受け付ける内容

- バグ報告（再現手順・期待結果・実結果を含む）
- セキュリティ改善（公開Issueで扱えない内容は private 連絡）
- ドキュメント改善
- 既存機能の品質改善（テスト追加、型安全性改善、性能改善）

## 受け付けない内容

- `LICENSE` の条件に反する再配布・商用利用を前提にした変更
- 個人情報保護や医療情報ガイドラインへの準拠を弱める変更
- 大規模仕様変更（事前合意なし）

## Pull Request ルール

- 1PR = 1意図（機能追加・リファクタ・ドキュメント更新を混在させない）
- 変更理由を説明し、必要に応じてスクリーンショットやログを添付する
- 可能な限りテストを追加する
- `npm run test`, `npm run type-check`, `npm run lint` を通す

## セキュリティ報告

公開Issueでの報告が不適切な脆弱性は、リポジトリ管理者へ直接連絡してください。

## 行動規範

技術的な議論を歓迎しますが、人格攻撃・ハラスメント・差別的言動は許容しません。
