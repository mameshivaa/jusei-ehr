# クイックスタートガイド

V-OSS を初めて使う方向けの簡単な使い方ガイドです。

## 1. ログイン

1. ブラウザでアプリケーションを開く（例: `http://localhost:3000`）
2. メールアドレスとパスワードを入力
3. 「ログイン」ボタンをクリック

## 2. 患者を検索または登録

### 患者を検索する場合

1. ホーム画面から「患者管理」をクリック
2. 検索ボックスに患者名、フリガナ、電話番号、または患者IDを入力
3. 「検索」ボタンをクリック
4. 検索結果から患者を選択

### 新規患者を登録する場合

1. 患者一覧ページの右上「新規登録」ボタンをクリック
2. 必須項目（氏名、フリガナ）を入力
3. その他の情報を入力（任意）
4. 「保存」ボタンをクリック

## 3. 来院記録を追加

1. 患者詳細ページで「来院を追加」ボタンをクリック
2. 来院日時が自動的に記録されます

## 4. 施術記録を作成

1. 患者詳細ページの来院履歴から、来院日時をクリック
2. 「施術記録を追加」ボタンをクリック
3. SOAP形式で記録を入力：

   **S (主観的情報)**: 患者の訴え

   - 例: 「右肩が痛くて上がらない」

   **O (客観的情報)**: 所見

   - 例: 「右肩関節の可動域制限あり。外転60度まで」

   **A (評価・診断)**: 評価

   - 例: 「右肩関節周囲炎の疑い」

   **P (治療計画)**: 治療計画

   - 例: 「温熱療法とストレッチを実施」

4. 「保存」ボタンをクリック

## よく使う操作

### 患者を素早く検索する

- 患者IDで検索するのが最も速いです
- 患者名の一部でも検索できます

### 過去の施術記録を確認する

1. 患者詳細ページで来院履歴を確認
2. 来院日時をクリックして詳細を表示
3. 施術記録を確認

### 施術記録を修正する

1. 来院記録詳細ページで「編集」ボタンをクリック
2. 内容を修正
3. 「保存」ボタンをクリック
