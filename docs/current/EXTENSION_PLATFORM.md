## 2. 現在の実装状態

実装済み:

- コマンド寄与点

- 権限付与/剥奪
- 拡張状態の永続化
- 監査ログ連携

型定義中心:

- integrations
- exporters
- views

## 3. セキュリティ方針

- DB直アクセス禁止
- App API経由のアクセス
- 拡張権限 + RBAC の二重チェック

## 4. 運用上の注意

- 本番導入前に要求権限レビュー
- 外部通信先を明示許可
- 無効化手順とロールバックを準備
